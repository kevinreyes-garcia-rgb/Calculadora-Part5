# 🔮 Calculadora Científica Negra & Morada

Calculadora científica elegante con diseño dark mode en tonos púrpura y negro.

## ✨ Características

- Operaciones básicas y científicas (sin, cos, tan, ln, log, √, n!, π, e)
- Evaluación en tiempo real
- Soporte de teclado completo
- Diseño responsive y moderno
